LAYER 1 — ACOUSTIC SENSOR FIELD · 3RD DISTRICT · 180 NODES
📡
Earthworks M30 Mic Array
Pole-mounted nodes at signalized intersections and streetlight infrastructure. 600–900 ft coverage radius per node.
44.1kHz Sample Rate 140dB SPL 3Hz–40kHz
🖥️
Raspberry Pi 5 Edge Node
Runs PyAudio + librosa locally. Performs FFT, MFCC extraction, CNN inference. Only events above confidence threshold leave the node.
pyroomacoustics PyTorch CNN Local Inference
📶
4G/LTE Node Uplink
Each node transmits timestamped event packets to STDS Central via AT&T FirstNet — the same carrier network used by Cleveland CDP for mission-critical comms.
FirstNet Priority GPS-Synced NTP TLS 1.3
🎵
Audio Evidence Clip
On trigger: 10-second WAV clip (5s pre-event buffer + 5s post) captured and encrypted at the node. Uploaded to evidence store.
AES-256 SHA-256 Hash Chain of Custody
KAFKA EVENT STREAM · PROTOBUF PAYLOAD
LAYER 2 — STDS CENTRAL PROCESSING SERVER · ON-PREM OR AWS GOV CLOUD
⚙️
Apache Kafka Ingest
Message bus ingests node event streams. Guarantees ordering and zero dropped events under peak load. Each message includes node ID, timestamp, RPM estimate, confidence score, audio hash.
Topics: rpm-events node-heartbeat alerts
📐
GCC-PHAT Localizer
pyroomacoustics TDOA engine. When 3+ nodes trigger simultaneously, computes hyperbolic intersection to produce GPS lat/lon coordinate within ±25 feet. Output written to Redis.
3-node minimum ±25ft accuracy <2s latency
🔀
STDS API Gateway
FastAPI service exposes confirmed incidents as JSON over HTTPS. Authenticates outbound webhooks to Fusus and CAD. This is the integration bridge to the Cleveland RTCC.
REST + WebSocket Webhook Push OAuth 2.0
🗄️
PostgreSQL + Redis
Redis for rolling 60s RPM state per node. PostgreSQL stores confirmed incidents, audio evidence hashes, and GPS tracks for post-incident analytics and court export.
Hot state: Redis Cold: PostgreSQL CJIS Compliant

WHAT STDS PLUGS INTO (ALREADY IN CLEVELAND)
LIVE 📹
FLOCK SAFETY CAMERAS
3,400+ city-owned surveillance cameras accessible through Fusus. STDS incident geo-triggers automatic camera activation at event location.
LIVE 🔍
FLOCK LPR NETWORK
License plate reader network. STDS fires LPR corridor query on detection — capturing every plate present during the takeover window. Flock Raven also detects street takeovers independently; STDS adds acoustic confirmation.
LIVE 🗺️
AXON FUSUS MAP
Cleveland RTCC already runs FususONE. STDS posts events via the existing Fusus sensor webhook API — no new software needed at the RTCC. JSON payload over HTTPS with basic-auth token.
API 🚁
SKYDIO DFR DRONES
Fusus has a native Skydio drone integration. STDS-confirmed incidents can auto-trigger nearest Drone-as-First-Responder unit to fly to GPS coordinate before patrol arrives.
API 📟
CAD SYSTEM
Fusus pushes confirmed STDS events into Cleveland's CAD as structured CFS with incident type, address, and priority level — eliminating manual dispatcher entry.
API 📱
FususNOTIFY
RTCC's existing push notification system broadcasts STDS incidents to patrol MDTs, supervisor tablets, and traffic command — same channel used for all existing Fusus alerts.
WEBHOOK 📁
AXON EVIDENCE
Audio clips, GPS tracks, RPM timeline data, and LPR plates package into Axon Evidence automatically with chain of custody — ready for prosecutor review without manual assembly.
WEBHOOK 🏛️
SAFE SMART CLE
~300 privately-owned cameras opt-in to Fusus via CORE devices. STDS incident radius can trigger access requests to nearby private cameras through the existing SAFE SMART CLE permission framework.
STRATEGIC NOTE
Cleveland's RTCC already runs Axon Fusus — the same platform that natively ingests sensor webhooks from ShotSpotter, Skydio drones, Verkada cameras, and Flock LPR. STDS does not require a separate display system or new software at the RTCC. The integration is a single authenticated webhook endpoint: STDS pushes a JSON incident payload to Fusus, and every existing downstream workflow — camera activation, CAD entry, officer notification, drone dispatch, evidence packaging — fires automatically through infrastructure Cleveland already owns and operates.
